# NFL Super Bowl Champion Prediction

## Project Overview
This project aims to develop machine learning models that predict which NFL teams are most likely to win the Super Bowl based on regular season performance statistics.

Using historical NFL team data and Super Bowl results, we will compare multiple classification models such as Logistic Regression, Random Forest, Gradient Boosting, and XGBoost to identify the characteristics of championship teams.

## Objectives
- Collect and preprocess NFL team statistics and Super Bowl winner data
- Perform exploratory data analysis and feature engineering
- Build baseline and advanced machine learning models
- Evaluate model performance using classification metrics
- Rank teams by predicted championship probability
